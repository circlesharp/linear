### 6. 向量运算的基本性质  
     1. 交换律  
        uT + vT = vT + uT  
     2. 结合律  
        (uT + vT) + wT = uT + (vT + wT)  
     3. 数量乘法分配律  
        k * (uT + vT) = k * uT + k * vT  
        (k + c) * uT = k * uT + k * vT  
     4. 数量乘法结合律  
        （k * c) * uT = k * (c * uT)  

### 7. 零向量
    1. O：对于任意一个向量 uT ，都存在一个向量 O ，满足：uT + O = uT  
        不去定义什么是零向量，而是从推导一个性质出发  
        O的维度由它在的空间中决定；每一个空间中有惟一的零向量  
        **注意**O没有箭头，本质是一个点，是没有方向的  
    2. -uT：对于任意一个向量 uT ，都存在一个向量 -uT ，满足：uT + -uT = O  
        -uT唯一（使用反证法去证明：假设存在另一个 vT ，也满足：uT + vT = O）  
        且: -uT = -1 * uT  
        **注意**两种重要的方法：1. 反证法，2. 数学归纳法

## 第三章_向量高级话题

### 1. 向量的规范化  
    1. 向量的模（向量的长度）  
        用 ||uT|| 表示（双竖线：二范数用双竖线表示/欧拉距离）  
    2. 向量的模的计算  
        uT = （u1, u2, ……, un)T  
        ||uT|| = (u1^2 + u2^2 + …… + un^2)^(1/2)  
    3. 单位向量(unit vector；向量的方向)  
        **重点**归一化/规范化/normalize：根据 向量uT 求 单位向量u_hat  
        **重点**单位向量长度已经不重要，方向才重要  
    4. 单位向量的计算  
        u_hat = uT / ||uT|| = (u1/||uT||, u2/||uT||, ……, un/||uT||)  
        u_hat = 1  
    5. 标准单位向量（Standard Unit Vector）  
        Standard Unit Vector 指向坐标轴的正方向  
        n维空间中，有n个 Standard Unit Vector  
        e1T = (1,0)  
        e2T = (0,1)

### 3. 向量的点乘与几何意义  
   1. uT.dot(vT) = u1*v1 + u2*v2 + …… + un*vn
   2. uT.dot(vT) = ||uT|| * ||vT|| * cos(Theta)
   3. 两个向量“相乘 (dot product)”，结果是一个数  
   4. 更严格的说法：点乘、内积

### 4. 向量点乘的直观理解
   1. 投影于其中一个向量上：uT.dot(vT) = ||uT|| * (||vT|| * cos(Theta) )
   2. 实质上，强调了向量的方向性；投影使得方向统一。
   3. 投影于坐标轴上：uT.dot(vT) = x1*x2 + y1*y2 + (x1*y2 + x2*y1)
   4. 但是(x1*y2 + x2*y1) = 0，因为cos(Theta)为0，彼此没有贡献。

### 6. 向量点乘的意义
    1. 求两向量的夹角: cos(Theta) = uT.dot(vT) / (||uT|| * ||vT||)  
        如果 uT.dot(vT) > 0, 夹角锐角；  
        如果 uT.dot(vT) = 0, 向量垂直，例如标准单位向量；  
        如果 uT.dot(vT) < 0, 夹角钝角。  
    2. 判断两个向量的相似程度（推荐系统）
        例如，电影推荐，每个电影都是一个多维空间中的一个点  
        判断每个电影向量的夹角： 
            锐角 => 相似（越大越相似）  
            直角 => 无关  
            钝角 => 背离（越小越背离）  
    3. 几何计算
        投影点的坐标：P(vT) = d * u_hat = (uT.dot(vT) / ||uT||) * u_hat

    1. 矩阵乘法不遵守交换律

## 第六章_线性系统

### 1. 线性系统与消元法
        1. 什么是线性 
            -> 未知数只能是一次方项
            -> 广义的，二维是直线，三维是直的平面
        2. 消元法的3个基本操作
            一个方程左右两边同时乘一个常数
            一个方程加（减）另一个方程
            交换两个方程的位置

### 2. 高斯消元法
        1. 增广矩阵
            在 系数矩阵 之后，增加一列 结果
        2. 高斯消元法 -> 让每一行的主元变成1
            使用增广矩阵：
            -> 矩阵某一行 * 一个常数
            -> 矩阵的一行 + 另一行
            -> 交换矩阵的两行
        3. 有 n 个未知数，一般就有 1 个方程。
        4. 主元(pivot，首元)，每一行的第一个非0元素
        5. 缺点：向前回代

### 3. 高斯-约旦消元法
        1. 相当于倒着使用高斯消元  
            -> 使得除了主元，其他都为0  
        2. 过程  
            1. 前向过程 -> 从上到下  
                选择最上的主元，化为1；  
                主元下面的所有元素都为0。  
            2. 后向过程 -> 从下到上  
                选择最下的主元；  
                主元上面的所有元素都为0。  

### 4. 【实现】高斯-约旦消元法
        **代码** LinearSystem.py

### 5. 线性方程组解的结构
        1. 解的结构
            1. 有唯一解
            2. 无解
            3. 有无数解
        2. 阶梯型矩阵
            行最简形式（RREF：reduced row echelon form):
                -> 非零行的主元位1，主元所在列(上下)的其他元素为0
                -> 就是实现 高斯-约旦消元法 之后的效果
        3. 重返解的结构
            1. 无解 （比较 A 与 RREF 的非零行）
                -> 系数矩阵非零行 < 行最简形式非零行
            2. 有解 （比较 A 与 RREF 的非零行，也就是方程的数量和未知数的数量）
                1. 有无数解 
                    -> 系数矩阵非零行 < 未知数个数
                2. 有唯一解 
                    -> 系数矩阵非零行 == 未知数个数

### 6. 直观理解线性方程组解的结构
        1. 方程的个数：
            n 个未知数，就需要n 个方程（n 个约束条件），才可能有惟一解
            从几何角度：
                相交得一个点就，惟一解
                相交得一条直线，无数解
                如果不相互相交成一个'something'（平面、直线、点），无解
        2. 从概率上说，如果方程（约束条件）个数比未知数多，更大可能没有解（因为碰巧相交太难了）。
        3. 对于RREF，从9种情况化简成2种/4种情况，很精彩
            结论  比较 A 与 RREF 的非零行；
                  比较 A 与 RREF 的非零行。

### 7. 更一般化的 高斯-约旦消元法
        含有主元的列 -> 主元列
        不含主元的列 -> 自由列
        最后，用主元列所代表的未知数作为等号的左边；自由列的未知数放在等号右边（向量的数量乘法）。

        2. 矩阵的逆和很多重要的命题连接在一起
            对于方阵A，则下面的4个命题等价
                -> A是非奇异矩阵（A可逆）
                -> 线性系统(齐次线性方程组)A*x = 0有唯一解，x = 0
                -> rref(A) = I
                -> A可以表示成一系列初等矩阵的乘积（A = E1**-1 * E2**-1 …… Ep**-1)
                -> 新命题： Ax=b 只有唯一解
        3. 拓展：证明一系列命题等价，使用“环状证明法”
            证明的过程很有意思

### 6. 矩阵的LU分解
        1. 数的分解 质因数分解（66 = 2*3*11）
        2. 一个矩阵也可以分解成几个矩阵乘积的形式，而且分解有不同的目的
            -> 矩阵LU分解，是以提高计算效率为目的
        3. 矩阵的LU分解
            A = L * U  （L: Lower Triangle Matrix/下三角矩阵； U: Upper Triangel Matrix/上三角矩阵）
            通常，分解的时候，L一般是“单位下三角矩阵”，而U不保证是“单位……”（因为，得到的U的主元不归一）
            ——————
            高斯消元法的过程，通过初等变换，将一个矩阵变成上三角矩阵
            Ep * … * E2 * E1 * A = U
                -> I * A = (E1**-1 * E2**-1 * …… * Ep**-1) * U
                -> L = (E1**-1 * E2**-1 * …… * Ep**-1)  # 如果没有出意外（不交换两行的位置）
        4. 为什么LU分解
            LU分解：O(0.5n**3)
            L*U*x = b; 设Ux=y, L*y=b; O(n**2)求出y; O(n**2) 求出x
            所以，共 O(0.5n**3)+O(n**2)
            ——————
            而，对比求解矩阵的逆 O(2*n**3)+O(n**2)

## 第八章_线性相关、线性无关和生成空间

### 1. 线性组合(线性代数的核心概念)
        1. 向量两个最基本的运算与线性组合
            向量的两个最基本的运算：向量加法 和 标量乘法
            着两个最基本的运算构建了线性代数中最重要的概念————线性组合
        2. 对于若干个n维向量 vec1, vec2, ……, vecp
            k1*vec1 + k2*vec2 + …… + kp*vecp 为这些向量的一个线性组合
            p.s. 可以类比线性方程，只是这里没有等于号
        3. 每一个向量，都可以写成单位向量的线性组合
            vec = (x, y, z) = x*e1 + y*e2 + z*e3
        4. 列视角下，矩阵和列向量的乘法可以写成线性组合
            -> 联系图形变换，将向量放到新的空间
        5. 在高斯消元中，矩阵里面每一个行都是一个线性组合

### 2. 线性相关和线性无关
        1. 线性相关的概念
            对于一个线性组合，存在一组不全为0的k，使得该线性组合为零向量，则称**这些向量**线性相关。
        2. 理解线性相关的等价表述：
            vec1, vec2, ……, vecn 线性相关 <=> 其中一个向量可以写成其他向量的线性组合
            即：如果某个k不为零，那这个向量可以写成其他向量的线性组合
            p.s. 判断齐次线性方程组，因为要么有唯一0解，要么有无穷多解。（详见355行）
        3. 线性无关
            对于一个线性组合，只有k全为0时，这个组合的结果为0向量，则**这些向量**线性无关。
            任何一个向量都不可以表示成为其他向量的线性组合。
            -> in other word，这些向量都非常重要，不能被其他向量替代！
            -> 我们会更加关心线性无关！

### 3. 矩阵的逆和线性相关、线性无关
        重要性质：
            m个n维向量，如果m>n，则这些向量线性相关。（个数大于维度）
                p.s. 齐次线性方程组求解问题。
                因为m>n，系数矩阵化为rref，肯定非零行<列数
            怎么线性无关呢？
                -> 有唯一零解：系数矩阵A可逆（必定m=n，因为可逆一定是方阵）
                -> 联系第7章第5节的等价5连：
                    矩阵A可逆(A是非奇异矩阵)
                    线性系统Ax=O只有唯一0解
                    rref(A) = I
                    A可以表示成一系列初等矩阵的乘积
                    Ax=b 只有唯一解
                    plus: 方阵A的列向量线性无关
                -> 从此，线性无关与前面的知识点联系起来了。
